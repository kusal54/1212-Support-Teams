Data One

Share your single data among branches

Details :-

  • Single bulk Broadband Package for all the branches of your business.
  • Data offers as Anytime packages.
  • A portal will be provided to monitor the data usage, manage the data bucket and manage data allocation for the branch access.
  • Portal will be created by the Account Manager.
  • Extra GB available (Standard Extra GB pricing would apply).
  • Static IP is to be provided with standard charges.
  • Data One service charge - Rs.100/- (per each branch).
  • Data is prorated in new connections.
  • Package upgrades & downgrades will be done through Account Managers.
  • Faults reported will be done as normal with the fauly branch User ID.

SLTM VoBox

Details :-

  • Similar to PBX but not complex as PBX.
  • Low cost than PBX
  • Any medium of Voice connectivity.
    - Megaline, FTTH and 4G LTE in any combination (2 to 50 connections)
  • Platinum category premium number.
    - Example: 0112XYXYXY / 0112XYZXYZ.
  • Interactive Voice Response (IVR).
    - Pre-recorded voice interacts with human to response the incoming call and direct the correct department/agent
  • Free calls within the group.
  • This feature is available in Both SLT And Mobitel.
  • CENTREX features.
    - Call Transferring, Call Forwarding, Call Conferencing, Short Code Dialing etc…

SLTM Univo

PBX and IOT Gateway Solution

  • IP PBX designed within the SLT R& D team with industry-leading technical features.
  • Connect wired or wireless IP extensions, PSTN phones, PC or mobile phone to create your internal PBX System.
  • Move from a Capex operation to an Opex operation through installments plan for your communication system.

Features:-

  • Mobile and Fixed Clients (SIP & GSM Connectivity).
  • Unlimited auto attendant/ IVR.
  • Flexible time-based call counting.
  • User management and group creation.
  • Ring groups.
  • Music-on-hold.
  • Customizable announcements.
  • Call Recording Reports.
  • Caller-ID.
  • Do-not-disturb.
  • Call forwarding.
  • Call waiting.
  • Call history and CDR.
